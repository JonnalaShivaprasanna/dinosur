Dinosaur Night Run: A Night-time themed version of Google's No Internet Game. Press the space bar to start the game. Then use the up (↑) and down (↓) arrow keys to control the dinosaur.

Dinosaur Night Run

Google Chrome’s hidden No Internet Game has been around since 2014, and it's received a few upgrades since its launch. Our team at nointernetgame.net reskinned this classic game with various different characters (always adding more). This version is the same character as the traditional No Internet Game, the only difference is the Night time theme.
